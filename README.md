# Ultralight Orchestration Example
This repo is a spin-off of Burke Holland's Ultralight Orchestration demo and video.

This contains a minimal multi-agent system with an orchestrator, a planner, a coder, and a designer working together providing orchestration between Claude, Codex and Gemini.

Visit [this gist](https://gist.github.com/burkeholland/0e68481f96e94bbb98134fa6efd00436)
  for Burke's detailed breakdown of the architecture, design decisions, and implementation details.  You can click on the `Install` links in the gist and it will create the agents directly in your own VS Code instance.

Watch this [Youtube video](https://www.youtube.com/watch?v=-BhfcPseWFQ) for Burke's walkthrough of the code and a demo of the agents in action.
