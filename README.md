# Burke Holland's Ultralight Orchestration
A minimal multi-agent system with an orchestrator, a planner, a coder, and a designer working together providing orchestration between Claude, Codex and Gemini.


Visit this gist for a detailed breakdown of the architecture, design decisions, and implementation details.  You can click on the `Install` links to create the agents directly in your own VS Code instance:
